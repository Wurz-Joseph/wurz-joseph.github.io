Changing CSS Properties Using JavaScript

"The HTMLElement.style property is used to get as well as set the inline style of an element. When getting, it returns a CSSStyleDeclaration object that contains a list of all styles properties for that element with values assigned for the attributes that are defined in the element's inline style attribute."
-MDN web docs

ONE

This code is used to specify only one CSS property at a time.

//div one
document.querySelector(".one").style.color = "blue";
document.querySelector(".one").style.backgroundColor = "gray";
TWO

This code block gives an example of specifying multiple properties at once.

//div two
document.querySelector(".two").style.cssText = "color: white; background-color: orange; font-size: 2em; border-radius: 15px;";
THREE

This is an example of changing the CSS property with an event listener.

//div three
btn.onclick = function () {
    document.querySelector(".three").style.cssText = "color: white; background-color: red; width: 50px; border-radius: 50%;"
}
