CSSANIMATIONS

"CSS animations make it possible to animate transitions from one CSS style configuration to another. Animations consist of two components, a style describing the CSS animation and a set of keyframes that indicate the start and end states of the animationâ€™s style, as well as possible intermediate waypoints"
-MDN Web docs. 

@keyframes slideIn {
    from {left: -150%;}
    to   {left: 0%;}
}

@keyframes fadeIn {
    from {opacity: 0;}
    to   {opacity: .75;}
}
.title{
    position: relative;
    font-size: 36px;
    text-align: center;
    font-weight: 900;
    margin: 15px 0;
    color: white;
    left: -150%;

    animation: slideIn 0.5s 0.5s forwards cubic-bezier(.5, .5, .5, 1.3);
}
.content{
    position: relative;
    color: black;
    font-weight: bold;
    font-size: 18px;
    margin: 15px 0;
    opacity: 0;
    
    animation: fadeIn 2.5s ease 1s normal forwards 1;

animation: fadeIn 2.5s ease 1s 1 normal forwards

  • animation-name - Specifies the name of the keyframe you want to bind to the selector
  • animation-duration - Specifies how many seconds or milliseconds an animation takes to complete
  • animation-timing-function - Specifies the speed curve of the animation
  • animation-delay - Specifies a delay before the animation will start
  • animation-iteration-count - Specifies how many times an animation should be played
  • animation-direction - Specifies whether or not the animation should play in reverse on alternate cycles
  • animation-fill-mode - Specifies what values are applied by the animation outside the time it is executing
  • animation-play-state - Specifies whether the animation is running or paused
