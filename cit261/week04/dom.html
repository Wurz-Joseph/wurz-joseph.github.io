DOM Manipulation

Create Element/Append Child

function addItem() {
    let newListItem = document.createElement("LI");
    let text = document.getElementById('itemText').value;
    if (text != '') {
      let textNode = document.createTextNode(text);
      newListItem.appendChild(textNode);
      document.getElementById('list').appendChild(newListItem);
      document.getElementById('itemText').value = '';
      document.getElementById('itemText').focus();
    }
  }

The function above uses document.createElement("LI") to create a new list item. Notice that "LI" is capitalized. The node names of HTML elements are always capitalized. The variable text is used to grab the user's entered text from the text box in my Grocery List program below. I added an if statement to prevent the user from being able to add empty list items to the grocery list.

HTML elments can consist of both an element node and a text node. The createTextNode method creates the text node portion which I then append to the newly create <li> element. Finally I used appendChild to add the <li> element with text to an ordered list.

Remove Child

function removeItem() {
  let list = document.getElementById('list');
  list.removeChild(list.lastChild);
}

The removeChild() method can be used to remove a specified child node of an element. In the example above I remove the last child node from the ordered list. You can also specify a specific item by giving the index of that item. You can do this by replacing the list.lastChild text with list.childNodes[i] where 'i' is the index of the node to be removed.

Insert Before

function insertAbove() {
  let newListItem = document.createElement("LI");
  let text = document.getElementById('itemText').value;
  let textNode = document.createTextNode(text);
  newListItem.appendChild(textNode);
  let list = document.getElementById('list');
  list.insertBefore(newListItem, list.childNodes[0]);

  document.getElementById('itemText').value = '';
  document.getElementById('itemText').focus();
}

The syntax is node.insertBefore(newnode, existingnode); newnode is where you would put the new element that will be inserted and existingnode is the node and index that your new element will be inserted in front of. In the example above I insert a new <li> element at the beginning of the list.
