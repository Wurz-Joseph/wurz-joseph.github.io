XMLHTTP Request

"Use XMLHttpRequest (XHR) objects to interact with servers. You can retrieve data from a URL without having to do a full page refresh. This enables a Web page to update just part of a page without disrupting what the user is doing. XMLHttpRequest is used heavily in AJAX programming."
-MDN

The following code block shows the basic code necessary to make an XMLHTTPRequest. Despite it's name, XMLHTTPRequest's can be used to retrieve any type of data, not just XML. In this example I use it to retrieve JSON data from a test API.

const xhr = new XMLHttpRequest();

xhr.open("GET", url, true);

xhr.send();
