CSS Transforms

CSS Transform Properties

  • translate(x,y,z) - Moves element along specified axis.
  • scale(x,y,z) - Defines a scale transformation
  • rotate(angle) - Defines a 2D rotation, the angle is specified in the parameter
  • skew(x-angle,y-angle) - Defines a 2D skew transformation along the X and the Y axis
  • matrix(n,n,n,n,n,n) - Defines a 2D transformation, using a matrix of all six 2D transform properties. The parameters are as follows: scaleX(), skewY(), skewX(), scaleY(), translateX(), translateY()

For this webpage I included a picture of a cloud and repeated the image three times. Each cloud is altered in some way using the transform property. I use transform in conjuction with CSS animations to try to make it look like it's behaving like a real cloud. Below are the transform properties used for each cloud.

  • Cloud 1 - translate, skew(65deg), rotateZ(20deg)
  • Cloud 2 - scale(1.5, 1.3), skewY(20deg)
  • Cloud 3 - scale(1.5, 2), translate, rotate(90deg) skew(30deg)
